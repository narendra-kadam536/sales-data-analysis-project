#  Sales Data Analysis using Python

##  Project Overview
This project analyzes monthly sales data of a retail store using Python.
The goal is to extract meaningful insights such as total sales per month,
best-selling products, and sales trends.

##  Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

##  Dataset
- sales_data.csv
- Columns: Month, Product, Quantity, Price

##  Analysis Performed
- Calculated total sales per month
- Identified the best-selling product
- Visualized sales trends using line and bar charts

##  Key Insights
- Sales vary across months showing seasonal trends
- Certain products consistently outperform others
- Visualizations help in quick business decision-making

##  How to Run
```bash
pip install -r requirements.txt
python sales_analysis.py
